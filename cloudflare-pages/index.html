Cloudflare Pages 免费托管这个静态入口页

完整 GEOFlow 后台是 Laravel/PHP 应用，需要 PHP 运行时、PostgreSQL、队列和常驻任务。 Cloudflare Pages 不运行 PHP 后端，因此后台本体需要部署到支持容器或 PHP 的服务。
